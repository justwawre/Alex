# NoSQL
Traditional database 可以看成是一张表,某一列的数据类型是确定的统一的。
NoSQL database 可以看成是一个JSON文件，每一行 

## 配合使用场景
* traditional database，如MySQL 存关键/基本信息，ACID，
* MongoDB 存附加文档，如图片，文档
* Redis 缓存热点数据。

# 支持的数据类型
* string
* list
* hash
* set
* sorted_set
